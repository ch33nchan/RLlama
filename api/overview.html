API Reference Overview

This module currently exposes the core classes you need to integrate RLlama:

RewardEngine

The central component that manages all reward components. It lets you add components, compute the aggregated reward, and inspect individual contributions.

from rllama import RewardEngine

# Create an engine
engine = RewardEngine()

# Register a component (any subclass of BaseReward)
engine.add_component(component)

# Compute the total reward for a given context
reward = engine.compute(context)

BaseReward

The base class for all reward components. To create your own, inherit from BaseReward and implement compute().

Module Status

At present, the API module provides these foundational classes. Other parts of RLlama—such as RewardComposer, RewardShaper, and the various built-in reward components—are defined in their respective submodules under rllama.rewards. Together, they form a complete reward-engineering toolkit.
