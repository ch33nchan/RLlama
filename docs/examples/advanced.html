Advanced Example: Automated Reward Optimization

RLlama provides a RewardOptimizer that can automatically tune your component weights by running multiple evaluation trials. Below is a complete example showing how to set it up:

from rllama import RewardEngine
from rllama.rewards.optimizer import RewardOptimizer

# 1. Initialize the reward engine and add your components
engine = RewardEngine()
engine.add_component(LengthReward(target_length=100))
engine.add_component(DiversityReward(history_size=5))
# ... add any other components or shaping you need ...

# 2. Create the optimizer
optimizer = RewardOptimizer(engine)

    engine.set_weights(weights)

# 4. Run the optimization loop
best_weights = optimizer.optimize(evaluate_weights, n_trials=100)

engine.set_weights(best_weights)
