Basic Example: Multi-Component Reward Function

Create a reward function that encourages helpful, safe responses of optimal length:

1

Create Configuration

config.yaml 
reward_components:
  - name: LengthReward
    params:
      target_length: 150
      strength: 1.0
  - name: DiversityReward
    params:
      history_size: 10
      similarity_threshold: 0.8

shaping_config:
  LengthReward:
    weight: 1.0
  DiversityReward:
    weight: 0.5
2

Initialize & Compute

basic_reward.py 
from rllama import RewardEngine

# Initialize engine with configuration
engine = RewardEngine('config.yaml', verbose=True)

# Define context for reward computation
context = {
    'response': 'RLlama is a modular reward engineering framework that makes it easy to design, test, and optimize reward functions for RL systems.',
    'query': 'What is RLlama?',
    'metadata': {
        'model': 'gpt-4',
        'temperature': 0.7
    }
}

# Compute reward
reward = engine.compute_and_log(context)
print(f'Total Reward: {reward:.4f}')

# Get component breakdown
component_rewards = engine.composer.calculate(context)
for component, value in component_rewards.items():
    print(f'{component}: {value:.4f}')

Our Philosophy: Reward Engineering as Software Engineering

Just as modern software development moved from monolithic applications to microservices, reward engineering should move from monolithic reward functions to composable reward systems.

🔧

Modularity

Each reward component has a single responsibility and can be tested independently

🔄

Reusability

Components work across domains - from robotics to language models to game AI

📊

Observability

Deep insights into reward component behavior with comprehensive logging

🎯

Optimization

Systematic hyperparameter tuning instead of manual trial-and-error

Neural Reward Model with Uncertainty

ensemble_example.py 
from rllama.models import EnsembleRewardModel
import torch

# Create ensemble with uncertainty quantification
model = EnsembleRewardModel(
    input_dim=768, 
    hidden_dims=[256, 128], 
    num_models=5
)

# Get predictions with uncertainty
state = torch.randn(1, 768)
reward, uncertainty = model(state, return_uncertainty=True)
print(f'Reward: {reward.item():.4f}, Uncertainty: {uncertainty.item():.4f}')
