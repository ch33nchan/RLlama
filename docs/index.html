About RLlama

RLlama is a Python library for designing, composing, and tuning reward functions in RL. Mix components, auto-search weights, and visualize term contributions.

Composable

Mix and match reward components to build complex objectives.

Optimizable

Auto-search over weights to maximize performance.

Explainable

Visualize contributions of each reward term.
