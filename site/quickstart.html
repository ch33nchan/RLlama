Quickstart

This guide will walk you through a basic example of using RLlama to define and manage a reward function for a simple agent.

1. Define Reward Components

First, let's create some basic reward components. These are simple Python functions that will calculate a reward based on the agent's state or action. 

from rllama.rewards import component

@component
def goal_distance(state, goal):
    """Calculates reward based on distance to a goal."""
    distance = abs(state - goal)
    return 1.0 / (1.0 + distance)

@component
def action_penalty(action
):
    """Applies a small penalty for every action taken."""
    return -0.05 * abs(action)
